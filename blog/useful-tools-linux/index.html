Daily Work Tools (Office Tools, etc.)

Web Browser

  1. Tor (online anonymity)
  2. Google Chrome
    • Vimimum, Vrome (Vim-like key bindings)
    • AdBlock Plus (block ads)
    • Pocket (save pages to read later)
    • Send from Gmail (send the link of current page via Gmail)
    • feedly (RSS feed)
    • TV Clicker
    • PrintFriendly (print web pages to PDF)
    • Secure SSH (SSH client in Google)
  3. Firefox/Iceweasel
    • Vimperator (Vim-like key bindings, the ultimate plugin for Firefox)
    • AdBlock Plus (block ads)
    • NoScript (block scripts on website which makes web surfing faster and securer)
    • Pocket (save pages to read later)
    • Flashblock (block flash vedio on websites which makes web surfing faster and securer)
    • Shareaholic (the ultimate way to share webpages with yourself/friends)
    • Xmarks (a standalone synchronization plugin for Firefox)
    • feedly (RSS feed)
    • FireSSH (SSH client for Firefox)
    • firebug (the ultimate way to modify/hack websites)
    • Greasemonkey/scriptish
    • cliget
      Generates curl/wget commands that emulate the request as though it's coming from your browser allowing you to download protected files directly to a separate machine.
    • lget
      Allows you to initiate a direct download of a file given a url.
    • DownThemAll
      The first and only download manager/accelerator built inside Firefox!
    • PrintFriendly
    • Open in Chrome
    • IETab
    • Live HTTP Headers
      View HTTP headers of a page and while browsing.
  4. w3m (a powerful text-based web browser)
  5. lynx (another text-based web browser)
